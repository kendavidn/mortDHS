Gestating R package for extracting mortality estimates from DHS data

Install with: 

```
if (!require(pacman)) install.packages("pacman")
pacman::p_load_gh("kendavidn/mortDHS")
```

The PDF manual can be found [here](https://github.com/the-graph-courses/misc/blob/main/mdhs_0.1.0_manual.pdf). 

Vignette can be found [here](https://demolandscapes.info/wp-content/uploads/2022/07/mdhs-Vignette-1.pdf)

Visit [demolandscapes.info/](https://demolandscapes.info/) for a demo of outputs from the package.
